解析vue-cli request.js拦截过滤器

理论分析：
    request.js的原理非常简单 整个流程就是一个拦截器
把axios原有的东西做了一次封装在发送请求之前配置点东西(请求头之类的)拦截到返回请求的时候做一系列判断要不要返回到页面，
同时也可以处理错误code，减少了代码的重复量，有错误就在request.js 中给你弹框了，就不必再每个页面去判断一次错误去弹框
